Adds custom fields for the company costumers to the WooCommerce checkout and allows Czech companies to fill company credentials from the API.

== Description ==
Adds Company & VAT numbers (IČO & DIČ) to WooCommerce billing fields. When billing to Czech republic, you can autofill fields Company, VAT number, Address, City, and Postcode based on IČO. Source of the data is ARES application API, more info can be found on the website of [Ministry of Finance](http://wwwinfo.mfcr.cz/ares/ares.html.en) or in the [ARES API documentation](http://wwwinfo.mfcr.cz/ares/ares_xml.html.en).

Credit for the plugin icon goes to [Creaticca Ltd](https://www.iconfinder.com/bendavis).

== Installation ==
Just follow the standard [WordPress plugin installation procedure](http://codex.wordpress.org/Managing_Plugins). The plugin does not have any Settings page.
